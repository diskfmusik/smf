
// トラックチャンクの終わり
ff 2f 00

// テンポ
ff 51 03 (3 byte)
四分音符の長さをマイクロ秒単位で表現
if (bpm == 120) 60*10⁶ / 120 = 500000(μsec)


// 拍子
ff 58 04 nn dd cc bb
nn = 分子
dd = 分母(二のマイナス乗で表す) 2:四分音符 3:八分音符 4:十六分音符
cc = メトロノーム間隔(四分音符間隔なら0x18(=24))
bb = 四分音符あたりの三十二分音符の数

// キー(調)
ff 59 02 sf mi
sf > 0 シャープの数, sf < 0 フラットの数
mi == 0 長調 mi == 1 単調
